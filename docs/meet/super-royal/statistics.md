---

**Example request**:
```plain
GET https://meet.trackmania.nadeo.club/api/me/super-royal/stats
```

**Example response**:
```json
{
    "masterWon": 0,
    "goldWon": 1,
    "silverWon": 2,
    "bronzeWon": 5
}
```
